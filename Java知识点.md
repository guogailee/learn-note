## Java注解

- 元注解

  - Documented：标记生成javadoc
  - Inherited：标记继承关系
  - Retention：注解的生存期
  - Target：标注的目标

- 标记注解

  - Override：覆盖超类中的方法
  - Deprecated：标记为丢弃的类
  - SuppressWarnings：抑制编译器发出的警告

- 自定义注解：配合反射使用

  ```
  XXX xxx = (XXX)class.getAnnotation(XXX.class);
  ```
